# 生产力书籍

欢迎来到生产力书籍网站！这里汇集了关于生产力、学习、习惯养成的系统化方法和实践指南。

## 📚 已发布书籍

### [Thomas Frank Book - 系统化生产力](./docs/thomas-frank/intro.md)

基于 Thomas Frank 的内容整理，教你建立系统化的生产力系统。

**核心内容**：
- 系统思维
- 计划和目标管理
- 深度专注
- 学习方法
- 能量管理
- 数字生活工具

**状态**: ✅ 全书完稿（12章）

### [Feel-Good Productivity - 让生产力成为感觉良好的来源](./docs/feel-good/intro.md)

一种全新的生产力方法，让你在做重要事情时感觉良好。

**核心内容**：
- 生产力心理学
- 时间管理系统
- 专注和深度工作
- 能量管理
- 社交和关系
- 持续改进

**状态**: 🔄 持续更新中（24/32章）

## 🎯 为什么选择这本书？

### 系统化方法
不是碎片化的技巧，而是完整的系统。

### 科学依据
基于心理学和神经科学研究。

### 可执行
